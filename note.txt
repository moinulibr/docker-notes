প্রজেক্ট এনভায়রনমেন্ট সেটআপ গাইড: ডকার (Docker) ব্যবহার
এই গাইডটি ব্যাখ্যা করবে ডকার কী, কেন আমরা এটিকে লোকাল ডেভেলপমেন্টের জন্য ব্যবহার করছি, এবং উইন্ডোজ অপারেটিং সিস্টেমে কীভাবে এটি সেটআপ করে প্রজেক্ট চালু করতে হয়।

১. ডকার কী? (What is Docker?)
ডকার হলো কন্টেইনারাইজেশন (Containerization) প্রযুক্তি ব্যবহার করে অ্যাপ্লিকেশন প্যাকেজিং এবং বিতরণ করার একটি প্ল্যাটফর্ম।

সহজ কথায়, ডকার আপনার অ্যাপ্লিকেশন এবং তার প্রয়োজনীয় সমস্ত পরিবেশ (যেমন: পিএইচপি, মাইএসকিউএল, নোড.জেএস) একটি ছোট, বিচ্ছিন্ন প্যাকেজ বা কন্টেইনারে মুড়িয়ে দেয়।

কেন ডকার ব্যবহার করবেন?
আমরা XAMPP/Laragon এর পরিবর্তে ডকার ব্যবহার করছি কারণ এটি পরিবেশের অমিল (Environment Mismatch) সমস্যা দূর করে।

সমস্যা

Laragon/XAMPP (ভার্সন কনফ্লিক্ট)

Docker (কন্টেইনার)

পরিবেশের স্থিরতা

আপনার ল্যাপটপে PHP 8.1, কিন্তু সার্ভারে PHP 8.3। ফলে কোড ব্রেক করতে পারে।

আমরা কন্টেইনারে PHP 8.3 সেট করব। এই কন্টেইনারটি আপনার বা আমার বা লাইভ সার্ভার—সব জায়গায় হুবহু একই আচরণ করবে।

পরিষ্কার সিস্টেম

লোকাল মেশিনে সরাসরি সার্ভার সফটওয়্যার ইনস্টল হয়।

কন্টেইনার একটি বিচ্ছিন্ন পরিবেশে চলে। এটি আপনার উইন্ডোজ সিস্টেমের কোনো সেটিং পরিবর্তন করে না।

টিমওয়ার্ক

একজন ডেভেলপারের পরিবেশে কাজ করা কোড অন্যজনের মেশিনে না-ও চলতে পারে।

ডকার ফাইলগুলো গিটে কমিট করা থাকে। টিমের সবাই একই ফাইল ব্যবহার করে একই পরিবেশ তৈরি করতে পারে।


২. উইন্ডোজে ডকার সেটআপ প্রক্রিয়া (Windows Setup Process)
উইন্ডোজে ডকার ব্যবহারের জন্য Docker Desktop এবং WSL 2 (Windows Subsystem for Linux 2) অপরিহার্য।

ধাপ ১: WSL 2 (Windows Subsystem for Linux 2) ইনস্টল করা
Docker Desktop চালানোর জন্য Windows 10 (ভার্সন 2004 বা তার পরের) বা Windows 11 এ WSL 2 থাকা আবশ্যক।

১. PowerShell চালু করুন: উইন্ডোজ স্টার্ট মেনুতে PowerShell সার্চ করে "Run as Administrator" হিসেবে ওপেন করুন।
২. WSL ইনস্টল কমান্ড: এই কমান্ডটি দিন (এটি Linux কার্নেল ও Ubuntu ডিস্ট্রিবিউশন স্বয়ংক্রিয়ভাবে ইনস্টল করবে):

wsl --install

৩. কম্পিউটার রিস্টার্ট: ইনস্টলেশন শেষ হলে আপনার কম্পিউটার রিস্টার্ট করুন।

ধাপ ২: Docker Desktop ইনস্টল করা
১. ডাউনলোড: অফিসিয়াল ওয়েবসাইট থেকে Docker Desktop for Windows ডাউনলোড করুন।
২. ইনস্টল করুন: .exe ফাইলটি রান করুন। ইনস্টলেশনের সময় নিশ্চিত করুন যে 'Use WSL 2 instead of Hyper-V' অপশনটি চেক করা আছে।
৩. Docker চালু নিশ্চিতকরণ: ইনস্টল করার পরে Docker Desktop অ্যাপ্লিকেশনটি চালু করুন। সিস্টেম ট্রে (System Tray)-তে ডকার আইকনটি সবুজ বা রানিং স্টেট দেখানো পর্যন্ত অপেক্ষা করুন।
